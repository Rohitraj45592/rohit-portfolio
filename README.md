# Rohit Raj — Portfolio 💼

> Personal developer portfolio built with React, TypeScript & Framer Motion

## 🚀 Live Demo

**[👉 Click here to view live](https://rohit-portfolio-mu-ochre.vercel.app/)**

## 🧠 What it is

A fully custom, animated single-page portfolio showcasing my projects, skills, education, and journey as an AI & full-stack developer — with an AI-powered search assistant built right in to answer questions about my work.

## 🏗️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + TypeScript + Tailwind CSS |
| Routing | TanStack Router |
| Animation | Framer Motion |
| Build Tool | Vite |
| Deployment | Vercel |

## ✨ Features

- 🎬 Animated hero with live availability badge
- 🧩 Filterable skill grid (Languages, Frameworks, AI/GenAI, Tools)
- 🗂️ Project case studies with live device previews
- 🎓 Education & achievements timeline
- 🤖 AI search assistant to explore my portfolio
- 🌗 Dark / Light theme toggle
- 📱 Fully responsive design
