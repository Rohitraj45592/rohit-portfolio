# Rohit Raj — Portfolio

Live: [rohit-portfolio-mu-ochre.vercel.app](https://rohit-portfolio-mu-ochre.vercel.app/)

A personal developer portfolio showcasing my projects, skills, education, and experience — built as a fully custom, animated single-page React application.

## ✨ Features

- Animated hero section with live availability badge
- Interactive skill grid with category filters (Languages, Frameworks, AI/GenAI, Tools, Databases)
- Project timeline with deep-dive case studies and live device previews
- Education and achievements timeline
- AI-powered search assistant to explore the portfolio content
- Dark/Light theme toggle
- Fully responsive across desktop, tablet, and mobile

## 🛠 Tech Stack

| Category | Technology |
| --- | --- |
| Framework | React + TypeScript |
| Routing | TanStack Router |
| Build Tool | Vite |
| Styling | Tailwind CSS v4 |
| Animation | Framer Motion |
| Icons | Lucide React |
| Deployment | Vercel |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Rohitraj45592/rohit-portfolio.git
cd rohit-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:5175](http://localhost:5175) to view it locally.

## 📂 Project Structure

```
src/
├── routes/        # Page routes (file-based routing via TanStack Router)
├── components/     # Reusable UI components
├── data/           # Project and content data
└── styles.css      # Global theme and design tokens
```
